## 📋 Overview

<a href="#graphPadPrism">GraphPad Prism</a> is a scientific graphing and statistical analysis application designed specifically for life scientists — combining the statistical analyses that biomedical research requires with publication-quality graph creation in a single application that guides researchers through statistical choices rather than assuming statistical expertise.

The <a href="#graphPadPrism">statistical analysis breadth</a> covers the full range of analyses that life science research requires — t-tests, one-way and two-way ANOVA with multiple comparison tests, linear and nonlinear regression, survival analysis, dose-response curves, contingency table analysis, correlation, and many others. Each analysis comes with built-in guidance about when to use it, what the results mean, and how to interpret the statistical output — making the statistical tools accessible to researchers whose primary expertise is biology rather than statistics. The <a href="#graphPadPrism">nonlinear curve fitting</a> is Prism's most technically distinctive capability — fitting experimental data to any equation from an extensive built-in library of pharmacological, biological, and mathematical models, or user-defined equations. Dose-response curves, enzyme kinetics, receptor binding, pharmacokinetic models, and growth curves all fit through Prism's curve fitting engine with the parameter estimation and confidence interval reporting that publication requires.

The <a href="#graphPadPrism">data organization in spreadsheets</a> uses a structured data entry format specifically designed for the types of experimental data life scientists collect — side-by-side replicates, nested data structures, multiple treatment groups — with data entry formats that match experimental designs rather than requiring researchers to adapt experimental data to a generic spreadsheet structure. The <a href="#graphPadPrism">publication-quality graph output</a> produces figures that meet the visual and technical standards of scientific journals directly from Prism — without requiring a separate graphics application to make graphs look publication-ready.</p>

  <img src="https://www.graphpad.com/guides/prism/latest/user-guide/images/clip0024.png" alt="GraphPad Prism screenshot"/>

<a href="#graphPadPrism">Multiple comparison tests</a> handle the post-hoc analysis that ANOVA results require — Tukey, Dunnett, Bonferroni, Holm-Sidak, and other corrections applied automatically after ANOVA with results displayed in the graph as significance stars or bars. <a href="#graphPadPrism">Error bar options</a> include standard deviation, standard error of the mean, confidence intervals, and other measures of variability for the appropriate representation of data variability in scientific figures.

<a href="#graphPadPrism">Interpolation from standard curves</a> calculates unknown concentrations from standard curve fits — the ELISA, standard curve, and calibration analysis workflow that analytical biology depends on, integrated directly into the Prism workflow. <a href="#graphPadPrism">Live link between data and graphs</a> updates graphs automatically when data changes — editing data in the spreadsheet updates the graph, the statistical analysis, and all derived values simultaneously for accurate final figures when data is corrected or updated.</p>
